Getting Started

First Launch

When you first open Byblos, an onboarding wizard guides you through:

  1. Microphone permission — Byblos needs mic access to hear you. Grant it when prompted.
  2. Download a model — Pick a Whisper model. Whisper Base (142 MB) is recommended for most users.
  3. You're ready — Byblos appears as a waveform icon in your menu bar.

Permissions

Byblos may request these macOS permissions:

  • Microphone — required for voice recording
  • Accessibility — required for typing text into other apps and for the hold-to-record hotkey. Grant in System Settings → Privacy & Security → Accessibility.

Recording & Transcription

Single-Click Recording

Left-click the Byblos icon in the menu bar to start recording. Left-click again to stop. That's it.

While recording:

  • The menu bar icon turns red
  • A floating overlay appears at the bottom of the screen showing streaming text
  • Word count and elapsed time display in the overlay

Hold-to-Record Hotkey

Hold Option (or your configured key) to record, release to stop. This is the fastest way to use Byblos — no clicking needed.

Configure the hotkey in Settings → General → "Hold key to record" (Option, Control, or Fn).

Tip: The hotkey requires Accessibility permission. If it doesn't work, check System Settings → Privacy & Security → Accessibility.

Auto-Stop on Silence

Byblos automatically stops recording when you stop speaking (after ~3 seconds of silence). This is enabled by default.

Configure in Settings → General → "Auto-stop on silence" and adjust the silence delay (1–10 seconds).

Click the Overlay to Stop

You can also click the floating overlay itself to stop recording.

Output Modes

  • Type into active app (default) — transcribed text is typed directly into whatever app you were using
  • Copy to clipboard — text is copied to your clipboard instead. Configure in Settings → General → "Output mode".

Dictation Modes

Right-click the menu bar icon → Mode to switch. Byblos offers 7 modes:

Raw

Exact transcription with no processing. What you say is what you get.

Clean (default)

Removes filler words (um, uh, like, you know), fixes punctuation and capitalization. Best for general use.

Email

Professional tone with paragraph structure. Great for dictating emails — just speak naturally and it formats appropriately.

Notes

Converts your stream of consciousness into bullet points. Speak freely and get organized notes.

Code Comment

Concise, technical language prefixed with //. Strips filler and hedging language. Ideal when coding.

Translate

Speak in any language and get English text output. Uses Whisper's built-in translation capability. Supports 99+ languages.

Agent

AI assistant mode — speak commands instead of dictation. See Agent Mode below.

App-Aware Mode Switching

When enabled (Settings → General → "Auto-select mode based on app"), Byblos automatically picks the right mode based on which app you're in:

  • Mail, Gmail → Email mode
  • Slack, Discord, Messages → Clean mode
  • VS Code, Xcode, Cursor → Code Comment mode
  • Notes, Obsidian → Notes mode

Transcript Workspace

Right-click the menu bar icon → Show Transcripts to open the workspace.

Browsing

The sidebar lists all transcriptions grouped by date (Today, Yesterday, This Week, Earlier). Click an entry to view it in the detail panel.

Search

Use the search field to filter transcripts by content.

Recording from the Workspace

Click the large red record button at the bottom of the window. Partial text appears as you speak.

Editing

Click on transcript text in the detail panel to edit it directly.

Exporting

Toolbar → Export → choose Markdown, Plain Text, or JSON. Exports all transcripts or the selected one.

Undo

Say "scratch that", "undo that", or "delete that" to undo the last transcription. Also available in the right-click menu → Undo Last.

Agent Mode

Agent mode turns Byblos into a voice-controlled AI assistant that can interact with your Mac. It requires a local LLM to be installed.

Switch to Agent mode via right-click → Mode → Agent (brain icon).

What You Can Say

  • "What's on my screen?" — reads the content of your current window
  • "Find files called readme" — searches your computer via Spotlight
  • "Open Safari" — launches an application
  • "What's on my clipboard?" — reads clipboard content
  • "What time is it?" — answers using system context

How It Works

  1. Your voice is transcribed by Whisper
  2. The local LLM understands your intent and generates actions
  3. Actions are executed (file search, screen reading, AppleScript, etc.)
  4. Results are summarized and displayed or typed

All processing is local — nothing leaves your machine.

File Transcription

Byblos can transcribe audio and video files, not just live speech.

Drag and Drop

Drag an audio or video file onto the Transcript workspace. Supported formats: WAV, MP3, M4A, FLAC, OGG, MP4, MOV, MKV.

Import Button

In the Transcript workspace toolbar, click "Import Audio" to choose a file.

Non-WAV files are automatically converted using macOS's built-in audio converter.

Custom Vocabulary

Open Settings → Vocabulary to add custom word replacements.

Useful for:

  • Names that Whisper misspells ("jay bee" → "JB")
  • Abbreviations ("API" → "API" instead of "a p i")
  • Technical jargon specific to your work

Replacements are case-insensitive and applied after transcription. You can import/export vocabulary as JSON.

Local LLM Setup (Optional)

A local LLM enables AI-powered features: smarter dictation cleanup, Agent mode, and future capabilities. It runs in a separate process alongside Whisper.

Installing a Model

  1. Download a GGUF model. Recommended: Qwen2.5-7B-Instruct-Q4_K_M (~4.5 GB)
  2. Place it in ~/Library/Application Support/Byblos/llm-models/
  3. Restart Byblos — the LLM helper starts automatically

Tip: You need at least 8 GB of unified memory for a 7B model. Apple Silicon Macs with 16 GB+ work best.

Supported Models

Any GGUF-format model works. Recommended options:

  • Qwen2.5-7B-Instruct — best for structured output and Agent mode
  • Phi-3.5-mini (3.8B) — lighter, good for basic cleanup

License & Support

Troubleshooting

App won't record

  • Check microphone permission: System Settings → Privacy & Security → Microphone
  • Check that a model is downloaded: Settings → Models

Hotkey doesn't work

  • Grant Accessibility permission: System Settings → Privacy & Security → Accessibility → add Byblos
  • After rebuilding from source, you may need to re-grant permission

Transcription is empty

  • Speak closer to the microphone
  • Check Settings → Audio → Input device is correct
  • Try a larger model (Whisper Small or Medium) for better accuracy

Agent mode doesn't respond

  • Agent requires a local LLM. See Local LLM Setup.
  • Check the log file at ~/Library/Logs/Byblos.log for errors

Log File

Byblos logs to ~/Library/Logs/Byblos.log. Include this file when reporting bugs.
